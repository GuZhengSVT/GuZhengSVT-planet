<html><head></head><body><h2>前置条件</h2>
<ul>
<li>node 版本 &gt;= 18.18.0</li>
<li>pnpm 版本 &gt; 8.1.0</li>
</ul>
<h2>安装</h2>
<h3>克隆仓库</h3>
<p>登录 Github 账号，打开 <a href="https://github.com/lxchapu/astro-gyoza">lxchapu/astro-gyoza</a>，点击右上角的 Fork 按钮，将仓库克隆到你自己的账号下。</p>
<p>复制这个仓库的地址，打开终端，使用 <code>git clone</code> 命令将仓库克隆到本地。</p>
<blockquote>
<p>本项目推荐使用 pnpm 作为你的包管理器，如果你还没有安装 pnpm，请先安装 pnpm。</p>
</blockquote>
<h3>安装依赖</h3>
<pre><code>cd astro-gyoza
pnpm install
</code></pre>
<h3>命令介绍</h3>
<p>本地运行</p>
<pre><code>pnpm dev
</code></pre>
<p>打包静态文件</p>
<pre><code>pnpm build
</code></pre>
<p>本地预览</p>
<pre><code>pnpm preview
</code></pre>

<h2>部署</h2>
<blockquote>
<p>这里只介绍了 Vercel，你当然可以选择其他平台例如：Cloudflare Pages 或你自己的服务器。<br/>
部署之前，确保你已经修改 <code>site.url</code>。</p>
</blockquote>
<h3>部署到 Vercel</h3>
<p>登录 Vercel 账号，点击右上角的 Add new... 选择 Project。然后在 Import Git Repository 中选择刚刚 Fork 的仓库，点击 Import 按钮。</p>
<p>进入项目配置页面，直接点击 Deploy 按钮，静静等待部署完成就  了。</p>
<p>Vercel 会为你分配一个域名，你可以在项目设置中设置自定义域名，更多操作请参考 Vercel 文档。</p>
